Dicionário de Tags

Tags Utilizadas

<!DOCTYPE html>
Define o tipo de documento, que é HTML5.
<html>
Elemento raiz de um documento HTML. Contém todos os outros elementos.
<head>
Contém metadados sobre o documento, como o título e links para folhas de estilo.
<meta>
Define metadados como o charset e a viewport, essenciais para a responsividade e codificação do documento.
<link>
Usado para vincular recursos externos, como folhas de estilo e ícones.
<title>
Define o título da página, que aparece na aba do navegador.
<body>
Contém o conteúdo visível da página, como textos, imagens, links e outros elementos.
<header>
Define o cabeçalho da página, normalmente usado para navegação.
<nav>
Representa uma seção de navegação com links para outras partes do site.
<a>
Cria um link para outra página ou seção dentro da mesma página.
<div>
Define uma divisão ou seção genérica em um documento HTML.
<main>
Indica o conteúdo principal do documento.
<aside>
Usado para conteúdo relacionado, como uma barra lateral.
<img>
Exibe uma imagem no documento.
<dl>
Cria uma lista de termos e suas definições.
<dt>
Define um termo dentro de uma lista de definições.
<dd>
Descreve ou define o termo listado no elemento <dt>.
<section>
Define uma seção de conteúdo temático.
<h1> - <h3>
Define cabeçalhos, sendo <h1> o mais importante e <h3> de menor importância.
<p>
Define um parágrafo de texto.
<br>
Insere uma quebra de linha no texto.
<footer>
Define o rodapé da página, geralmente com links de contato e redes sociais.
<address>
Fornece informações de contato.
